## Installation

```bash
$ npm i
```

## Usage

Copy the example env file and make the required configuration changes in the .env file

For example:

```bash
$ cp env.example .env
```

## Running the app

First:

```bash
# development
$ npm start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

After:

```bash
$ npm run migration:run
```
